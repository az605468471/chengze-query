# 承泽 DeFi Shield 查询系统 - 资料包说明

## 📦 资料包内容

### 1. 前端代码 (`frontend/`)
- **技术栈**: React 18 + Vite + Tailwind CSS
- **主要功能**:
  - 合约地址查询分析
  - 风险评分系统（0-100分）
  - 多语言支持（中/英/日/韩）
  - 钱包连接（MetaMask/Trust Wallet等）
  - PDF报告导出
  - 支付套餐系统

### 2. 智能合约 (`contracts/`)
- **技术栈**: Solidity + Hardhat
- **合约功能**:
  - 链上查询记录
  - 查询工厂模式
  - BSC主网部署

### 3. 支付系统
- **收款钱包**: `0x105071cFde8Fbf8D9b2f869409d6380A42037685`
- **支持币种**: USDT/USDC/BNB (BEP-20)
- **套餐价格**:
  - 单次查询: $1.99
  - 10次套餐: $15 (省25%)
  - 50次套餐: $50 (省50%)

### 本地开发
```bash
# 安装依赖
cd frontend && npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 📋 主要文件说明

| 文件 | 说明 |
|------|------|
| `frontend/src/pages/Home.jsx` | 主页面，包含查询和结果展示 |
| `frontend/src/pages/Payment.jsx` | 支付套餐页面 |
| `frontend/src/components/Header.jsx` | 头部组件，包含钱包连接 |
| `frontend/src/components/SearchBar.jsx` | 搜索栏组件 |
| `frontend/src/components/RiskScore.jsx` | 风险评分组件 |
| `frontend/src/utils/api.js` | API工具函数，包含所有数据获取逻辑 |
| `contracts/contracts/DefiShieldQuery.sol` | 查询合约 |
| `contracts/contracts/DefiShieldQueryFactory.sol` | 查询工厂合约 |
