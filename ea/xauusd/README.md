## 🔧 策略逻辑

### 交易信号
```
多头信号（买入）：
1. 趋势向上（EMA20 > EMA50 > EMA200）
2. RSI超卖后反弹（<40）
3. MACD金叉或柱状图转正
4. 突破关键阻力位
条件：至少满足2个

空头信号（卖出）：
1. 趋势向下（EMA20 < EMA50 < EMA200）
2. RSI超买后回落（>60）
3. MACD死叉或柱状图转负
4. 突破关键支撑位
条件：至少满足2个
```

### 风险控制
```
手数计算：
1. 单笔风险 = 净值 × 风险比例
2. 止损金额 = 止损点数 × 每手价值
3. 手数 = 风险金额 / 止损金额
4. 动态调整：净值增长+20%，净值减少-30%

止损止盈：
1. 止损 = ATR × 1.5倍
2. 止盈 = ATR × 3倍（盈亏比1:2）
3. 移动止损：盈利30点后启动
4. 保本止损：盈利30点后设置
```

## 📈 自动优化

### 优化策略
1. **遗传算法**：模拟生物进化
2. **锦标赛选择**：选择最优个体
3. **交叉变异**：生成新参数
4. **适应度评估**：基于胜率、盈亏比、回撤

### 优化周期
- **每24小时**：检查一次是否需要优化
- **交易数量**：需要至少20笔交易数据
- **优化时间**：约5-10分钟
- **参数更新**：自动应用最优参数
