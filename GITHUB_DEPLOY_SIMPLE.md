# 承泽查询系统 - GitHub Pages 部署指南

## 🚀 快速部署步骤

### 第 1 步：注册 GitHub 账号
1. 打开 Chrome 浏览器
2. 访问：https://github.com
3. 点击 "Sign up"
4. 填写用户名、邮箱、密码
5. 完成注册

### 第 2 步：创建仓库
1. 登录 GitHub
2. 点击右上角 "+" → "New repository"
3. 仓库名称：`chengze-query`
4. 描述：承泽查询系统
5. 选择 "Public"
6. 点击 "Create repository"

### 第 3 步：上传文件
1. 进入仓库页面
2. 点击 "Add file" → "Upload files"
3. 将以下文件拖拽上传：
   - `index.html`
   - `favicon.svg`
   - `icons.svg`
   - `assets/` 文件夹

### 第 4 步：启用 GitHub Pages
1. 进入仓库设置
2. 点击左侧 "Pages"
3. Source：选择 "Deploy from a branch"
4. Branch：选择 "main"
5. 文件夹：选择 "/ (root)"
6. 点击 "Save"

### 第 5 步：访问网站
1. 等待 1-5 分钟
2. 访问：`https://您的用户名.github.io/chengze-query/`
3. 在 Chrome 浏览器打开

---

## 📦 项目文件清单

需要上传的文件：
- `index.html` - 主页面
- `favicon.svg` - 网站图标
- `icons.svg` - 图标文件
- `assets/` - 静态资源文件夹

---

## 🔧 常见问题

**Q：上传后网站打不开？**
A：检查文件结构，确保 `index.html` 在根目录。

**Q：需要多长时间？**
A：通常 1-5 分钟。

**Q：可以自定义域名吗？**
A：可以在 Pages 设置中配置。
