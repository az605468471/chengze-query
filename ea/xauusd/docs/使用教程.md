# XAUUSD EA 使用教程

## 📋 EA配置

### 基础配置
- **交易品种**：XAUUSD（现货黄金）
- **平台**：MT4/MT5
- **时间周期**：H1（1小时）
- **资金规模**：$2000-$10000
- **风险偏好**：稳健型

### EA策略
- **策略类型**：趋势跟踪 + 关键位突破
- **核心逻辑**：
  1. **趋势判断**：EMA均线系统判断趋势方向
  2. **关键位识别**：自动识别支撑阻力位
  3. **入场信号**：突破关键位 + 趋势共振
  4. **出场策略**：移动止损 + 目标位止盈
  5. **仓位管理**：根据风险动态调整仓位

### 技术指标
- **趋势指标**：EMA(20, 50, 200)
- **动量指标**：RSI(14), MACD(12,26,9)
- **波动指标**：ATR(14)
- **支撑阻力**：自动识别关键价位

### 风险控制
- **单笔止损**：账户资金的2%
- **最大回撤**：10%
- **最大持仓**：同时最多2单
- **交易时间**：欧盘+美盘（北京时间15:00-24:00）

## 🚀 快速开始

### MT4安装
```bash
1. 复制 MQL4/XAUUSD_EA.ex4 到 MT4的 Experts 目录
2. 重启MT4
3. 在"导航"面板找到 XAUUSD_EA
4. 拖拽到 XAUUSD H1 图表
5. 设置参数，点击"确定"
```

### MT5安装
```bash
1. 复制 MQL5/XAUUSD_EA.mq5 到 MT5的 Experts 目录
2. 编译EA（按F7）
3. 在"导航"面板找到 XAUUSD_EA
4. 拖拽到 XAUUSD H1 图表
5. 设置参数，点击"确定"
```

## 🔧 参数说明

### 核心参数
| 参数 | 默认值 | 说明 |
|------|--------|------|
| LotSize | 0.01 | 基础手数 |
| RiskPercent | 2 | 单笔风险(%) |
| MaxDrawdown | 10 | 最大回撤(%) |
| TakeProfit | 100 | 止盈点数 |
| StopLoss | 50 | 止损点数 |
| TrailingStop | 30 | 移动止损点数 |
| MaxPositions | 2 | 最大持仓数 |

### 交易时间
| 参数 | 默认值 | 说明 |
|------|--------|------|
| TradeStartHour | 15 | 开始时间(北京时间) |
| TradeEndHour | 24 | 结束时间(北京时间) |
| AvoidNews | true | 避开重大数据 |
