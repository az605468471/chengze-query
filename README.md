# DeFi Shield - 安全查询平台

## 简介
一键分析 DeFi 项目安全性，提供合约审计、持仓分布、流动性分析等功能。

## 功能
- 合约安全评分
- 持仓分布分析
- 流动性检查
- 多语言支持
- 钱包连接

## 使用方法
1. 输入合约地址 (0x...)
2. 点击分析
3. 查看风险评分和详细报告

## 技术栈
- React + Vite + Tailwind CSS
- ethers.js 链上查询
- 多语言 i18n

## 部署
运行 `npm run dev` 开始开发
运行 `npm run build` 构建生产环境

## API 密钥
需要申请 BSCScan API 密钥用于合约查询
